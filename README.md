# The Grand Hotel — Website

Marketing website for **The Grand Hotel**, Sunnyvale, CA (Silicon Valley).
This repository hosts the project to migrate the site off the current vendor
(Cendyn) onto a self-owned, Vercel-hosted stack and to modernize the UI/UX.

- **Live (current vendor) site:** https://www.svgrandhotel.com
- **Booking engine (separate vendor product):** https://svgrandhotel.reztrip.com

## Start here

See **[`docs/PROPOSAL.md`](docs/PROPOSAL.md)** for the full review of the
current site, the vendor lock-in analysis, the migration plan, and the
proposed rebuild.
